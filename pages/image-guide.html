Types of Images and their Description

JPEG

JPEG is a lossy compressed file format and makes it easier to store images at smaller size.

GIF

GIF is mostly used when using more less colors and large areas in an image.

PNG

PNG is a lossles compression file format, which is the easier choice to use it on the web. It is mostly used to store line drawings and text in a file
